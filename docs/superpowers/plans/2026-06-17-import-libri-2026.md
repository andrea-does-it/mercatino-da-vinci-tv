# Import Libri 2026 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Strumento riutilizzabile (pagina admin) che da una lista di ISBN recupera dati+copertina+prezzo da Libraccio, verifica la presenza a DB e importa i nuovi libri popolando `product` e installando le copertine; più, una tantum, la tabella arricchita dei libri 2026.

**Architecture:** Servizio di lookup ISBN isolato (`BookLookup`) che non dipende dallo stato dell'app; metodo `ProductManager::findByISBN` per la verifica; pagina admin che orchestra lookup→verifica→import riusando la pipeline immagini esistente (`ImageUtilities`). Sviluppo e test prima in `web/htdocs/staging/`, poi mirror in `web/htdocs/`.

**Tech Stack:** PHP (server-side, no runner locale), MySQL 5.7, GD (`ImageUtilities`), Bootstrap (admin UI esistente).

## Global Constraints

- **Commit**: mai riferimenti a Claude/AI nei messaggi di commit o PR.
- **Ambiente**: sviluppo e test PRIMA in `web/htdocs/staging/` (sito `https://www.comitatogenitoridavtv.it/staging`, DB separato). Mirror in `web/htdocs/` solo dopo i test su staging.
- **Esecuzione test**: nessun PHP/PHPUnit locale. Le unità di logica pura si verificano con una pagina di self-test PHP eseguita su staging; gli integration test (lookup runtime, insert, immagini) li esegue l'admin sull'URL di staging.

- **Prezzo mercatino**: `price = round(prezzo_listino / 2 - 1.50, 2)`. `prezzo_listino` conserva il prezzo di copertina pieno.

<?php
class BookLookup {
  public static function validateIsbn13($isbn) {
    $isbn = preg_replace('/[^0-9]/', '', (string)$isbn);
    if (strlen($isbn) !== 13) return false;
    $sum = 0;
    for ($i = 0; $i < 12; $i++) {
      $sum += (int)$isbn[$i] * ($i % 2 === 0 ? 1 : 3);
    }
    $check = (10 - ($sum % 10)) % 10;
    return $check === (int)$isbn[12];
  }
}

```php
  public static function coverUrl($isbn) {
    $isbn = preg_replace('/[^0-9]/', '', (string)$isbn);
    return "https://www.libraccio.it/images/{$isbn}_0_500_0_75.jpg";
  }

  public static function lookup($isbn) {
    $isbn = preg_replace('/[^0-9]/', '', (string)$isbn);
    if (!self::validateIsbn13($isbn)) return null;

    $warnings = [];
    $html = self::httpGet("https://www.libraccio.it/src/?FT={$isbn}&ch=libraccio");
    if ($html === null) { $warnings[] = 'Libraccio non raggiungibile'; }

    $title = $authors = $publisher = '';
    $listPrice = null;
    if ($html !== null) {
      $title     = self::extractTitle($html);
      $listPrice = self::extractPrice($html);
      // autori/editore: best-effort dal markup; vuoti se non trovati
      $authors   = self::extractMeta($html, 'autori');
      $publisher = self::extractMeta($html, 'editore');
      if ($title === '')     $warnings[] = 'Titolo non trovato';
      if ($listPrice === null) $warnings[] = 'Prezzo non trovato';
    }

    return [
      'isbn'       => $isbn,
      'title'      => $title,
      'authors'    => $authors,
      'publisher'  => $publisher,
      'list_price' => $listPrice,
      'cover_url'  => self::coverUrl($isbn),
      'warnings'   => $warnings,
    ];
  }

  private static function httpGet($url) {
    $ctx = stream_context_create(['http' => [
      'timeout' => 12,
      'header'  => "User-Agent: Mozilla/5.0 (compatible; MercatinoBot/1.0)\r\n",
    ]]);
    $body = @file_get_contents($url, false, $ctx);
    return $body === false ? null : $body;
  }

  private static function extractPrice($html) {
    // cerca un importo in euro nel markup (es. "34,60" o "€ 34,60")
    if (preg_match('/(\d{1,3}(?:[.,]\d{2}))\s*€|€\s*(\d{1,3}(?:[.,]\d{2}))/u', $html, $m)) {
      $raw = $m[1] !== '' ? $m[1] : $m[2];
      return (float)str_replace(',', '.', $raw);
    }
    return null;
  }

  private static function extractTitle($html) {
    if (preg_match('/<title>(.*?)<\/title>/is', $html, $m)) {
      return trim(html_entity_decode(strip_tags($m[1])));
    }
    return '';
  }

  private static function extractMeta($html, $which) {
    return ''; // best-effort; raffinato in fase di test sui markup reali
  }
```

    $isbn = preg_replace('/[^0-9Xx]/', '', (string)$isbn);
